Perl CLDR

The Perl CLDR project aims to provide first-class support for the Unicode CLDR in Perl. We do this by supporting existing projects, writing new modules, writing documentation, adding features, fixing bugs, and upgrading to new CLDR versions. We work with Perl 5, Perl 6, and open source CLDR projects in other language communities.
